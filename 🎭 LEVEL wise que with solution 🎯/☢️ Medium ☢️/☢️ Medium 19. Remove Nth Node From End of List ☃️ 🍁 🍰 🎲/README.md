# Description 📜 ˋ°•*⁀➷

### Given the `head` of a linked list, remove the `nth` node from the end of the list and return its head.



</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 Input  ➤ head = [1,2,3,4,5], n = 2

  ### 📤 Output  ➤ [1,2,3,5]

</br>

  ### 📥 Input ➤ head = [1], n = 1

# Constraints 🔒 ˋ°•*⁀➷

🔹 **The number of nodes in the list is sz.** </br>
🔹 **1 <= sz <= 30** </br>
🔹 **0 <= Node.val <= 100** </br>
🔹 **1 <= n <= sz** </br>
