# Description 📜 ˋ°•*⁀➷

### Given a positive integer `n`, generate an `n x n` `matrix` filled with elements from `1` to `n`<sup>2</sup> in spiral order.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ n = 3

  ### 📤 `Output`  ➤ [[1,2,3],[8,9,4],[7,6,5]]

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤ n = 1

  ### 📤 `Output`  ➤ [[1]]

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 **`1 <= n <= 20`** </br>

</br>
