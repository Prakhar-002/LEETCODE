# Description рҹ“ң ЛӢВ°вҖў*вҒҖвһ·
### You are given two strings, `str1` and `str2`, of lengths `n` and `m`, respectively.
### A string `word` of length `n + m - 1` is defined to be **generated** by `str1` and `str2` if it satisfies the following conditions for each index `0 <= i <= n - 1`:
### &nbsp;&nbsp;&nbsp;&nbsp;вҖў If `str1[i] == 'T'`, the substring of `word` with size `m` starting at index `i` is **equal** to `str2`, i.e., `word[i..(i + m - 1)] == str2`.
### &nbsp;&nbsp;&nbsp;&nbsp;вҖў If `str1[i] == 'F'`, the substring of `word` with size `m` starting at index `i` is **not equal** to `str2`, i.e., `word[i..(i + m - 1)] != str2`.
### Return the **lexicographically smallest** possible string that can be generated by `str1` and `str2`. If no string can be generated, return an **empty string** `""`.

</br>

# Example рҹ’Ў 1пёҸвғЈ ЛӢВ°вҖў*вҒҖвһ·
  ### рҹ“Ҙ `Input`  вһӨ str1 = "TFTF", str2 = "ab"
  ### рҹ“Ө `Output`  вһӨ "ababa"
  ### рҹ”Ұ `Explanation`  вһӨ At indices `0` and `2` (T), substring of length `2` equals "ab". At indices `1` and `3` (F), substring "ba" вү  "ab". Both "ababa" and "ababb" are valid вҖ” return the lexicographically smaller "ababa".

</br>

# Example рҹ’Ў 2пёҸвғЈ ЛӢВ°вҖў*вҒҖвһ·
  ### рҹ“Ҙ `Input` вһӨ str1 = "TFTF", str2 = "abc"
  ### рҹ“Ө `Output`  вһӨ ""
  ### рҹ”Ұ `Explanation` вһӨ No string that satisfies all the conditions can be generated, so return `""`.

</br>

# Example рҹ’Ў 3пёҸвғЈ ЛӢВ°вҖў*вҒҖвһ·
  ### рҹ“Ҙ `Input` вһӨ str1 = "F", str2 = "d"
  ### рҹ“Ө `Output`  вһӨ "a"
  ### рҹ”Ұ `Explanation` вһӨ Since `str1[0] == 'F'`, the substring of length `1` must **not** equal "d". The lexicographically smallest character that is not "d" is "a".

</br>

# Constraints рҹ”’ ЛӢВ°вҖў*вҒҖвһ·
рҹ”№ `1 <= n == str1.length <= 10^4` </br>
рҹ”№ `1 <= m == str2.length <= 500` </br>
рҹ”№ `str1` consists only of `'T'` or `'F'`. </br>
рҹ”№ `str2` consists only of lowercase English characters. </br>

</br>
