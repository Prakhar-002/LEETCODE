# Description 📜 ˋ°•*⁀➷
### Given an array of positive integers `nums`, return an array `answer` that consists of the **digits of each integer** in `nums` after separating them in the **same order** they appear in `nums`.
### To **separate the digits** of an integer is to get all the digits it has in the same order.
### &nbsp;&nbsp;&nbsp;&nbsp;• For example, for the integer `10921`, the separation of its digits is `[1,0,9,2,1]`.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷
  ### 📥 `Input`  ➤ nums = [13,25,83,77]
  ### 📤 `Output`  ➤ [1,3,2,5,8,3,7,7]
  ### 🔦 `Explanation`  ➤ Separation of `13` → `[1,3]`, `25` → `[2,5]`, `83` → `[8,3]`, `77` → `[7,7]`. Combining in order gives `[1,3,2,5,8,3,7,7]`.

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷
  ### 📥 `Input` ➤ nums = [7,1,3,9]
  ### 📤 `Output`  ➤ [7,1,3,9]
  ### 🔦 `Explanation` ➤ The separation of each **single-digit** integer in `nums` is itself. So `answer = [7,1,3,9]`.

</br>

# Example 💡 3️⃣ ˋ°•*⁀➷
  ### 📥 `Input` ➤ nums = [10921,100]
  ### 📤 `Output`  ➤ [1,0,9,2,1,1,0,0]
  ### 🔦 `Explanation` ➤ Separation of `10921` → `[1,0,9,2,1]`, `100` → `[1,0,0]`. Combining in order gives `[1,0,9,2,1,1,0,0]`.

</br>

# Constraints 🔒 ˋ°•*⁀➷
🔹 `1 <= nums.length <= 1000` </br>
🔹 `1 <= nums[i] <= 10^5` </br>
