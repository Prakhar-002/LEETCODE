# Description 📜 ˋ°•*⁀➷
### Given a string `text`, you want to use the characters of `text` to form as many instances of the word **"balloon"** as possible.
### You can use each character in `text` **at most once**. Return the **maximum number of instances** that can be formed.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷
<img width="277" height="63" alt="image" src="https://github.com/user-attachments/assets/b157d3dc-abd1-4d6d-9006-d4cdb0d69e14" />

  ### 📥 `Input`  ➤ text = "nlaebolko"
  ### 📤 `Output`  ➤ 1
  ### 🔦 `Explanation`  ➤ The characters in `text` can form exactly **one** instance of "balloon" using `b, a, l, l, o, o, n`.

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷
<img width="480" height="72" alt="image" src="https://github.com/user-attachments/assets/0cbd51a1-d342-466d-acbb-e50994af24fc" />

  ### 📥 `Input` ➤ text = "loonbalxballpoon"
  ### 📤 `Output`  ➤ 2
  ### 🔦 `Explanation` ➤ The characters in `text` can form exactly **two** instances of "balloon" — enough `b`, `a`, `l` (×2 each), `o` (×2 each), and `n` are available.

</br>

# Example 💡 3️⃣ ˋ°•*⁀➷
  ### 📥 `Input` ➤ text = "leetcode"
  ### 📤 `Output`  ➤ 0
  ### 🔦 `Explanation` ➤ The string `text` is missing required characters like `b`, `a`, and `n`, so **zero** instances of "balloon" can be formed.

</br>

# Constraints 🔒 ˋ°•*⁀➷
🔹 `1 <= text.length <= 10^4` </br>
🔹 `text` consists of **lower case English letters** only. </br>

</br>
