# Description 📜 ˋ°•*⁀➷

### Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s.



</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### Input  ➤ s = "aab"

  ### Output  ➤ [["a","a","b"],["aa","b"]]

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### Input ➤ s = "a"

  ### Output  ➤ [["a"]]

</br>

# Constraints 🔒 ˋ°•*⁀➷

- **1 <= s.length <= 16**
- **s contains only lowercase English letters.**


</br>

# Topics 📋 ˋ°•*⁀➷

1. **String**
2. **Dynamic Programming**
3. **Backtracking**

</br>
