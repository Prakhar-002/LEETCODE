# Description 📜 ˋ°•*⁀➷

### Given `n` pairs of parentheses, write a function to *generate **all combinations** of well-formed parentheses*.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ n = 3

  ### 📤 `Output`  ➤ ["((()))","(()())","(())()","()(())","()()()"]

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤ n = 1

  ### 📤 `Output`  ➤ ["()"]

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 **`1 <= n <= 8`** </br>

</br>

# Topics 📋 ˋ°•*⁀➷

🔸 **Dynamic Programming**  </br>
🔸 **Backtracking**  </br>
