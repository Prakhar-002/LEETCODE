# Description 📜 ˋ°•*⁀➷

### Given two integers `a` and `b`, return the sum of the two integers without using the operators `+` and `-`.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ `a = 1, b = 2`

  ### 📤 `Output`  ➤ `3`

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ `a = 2, b = 3`

  ### 📤 `Output`  ➤ `5`

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 `-1000 <= a, b <= 1000` </br>

</br>
