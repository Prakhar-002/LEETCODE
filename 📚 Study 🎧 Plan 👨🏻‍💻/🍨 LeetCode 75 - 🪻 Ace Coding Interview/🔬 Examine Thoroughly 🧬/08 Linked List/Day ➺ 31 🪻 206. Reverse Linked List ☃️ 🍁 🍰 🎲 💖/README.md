# Description 📜 ˋ°•*⁀➷

### Given the `head` of a singly linked list, reverse the list, and return *the reversed list*.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ head = [1,2,3,4,5]

  ### 📤 `Output`  ➤ [5,4,3,2,1]

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤ head = [1,2]

  ### 📤 `Output`  ➤ [2,1]

</br>

# Example 💡 3️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤ head = []

  ### 📤 `Output`  ➤ []

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 **The number of nodes in the list is the range `[0, 5000]`.** </br>

🔹 **`-5000 <= Node.val <= 5000`** </br>

</br>
