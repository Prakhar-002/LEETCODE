# Description 📜 ˋ°•*⁀➷

### Given the `root` of a binary tree, return *the postorder traversal of its nodes' values*.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

  ### 📥 `Input`  ➤ root = [1,null,2,3]

  ### 📤 `Output`  ➤ [3,2,1]

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤ root = []

  ### 📤 `Output`  ➤  []

</br>

# Example 💡 3️⃣ ˋ°•*⁀➷

  ### 📥 `Input` ➤  root = [1]

  ### 📤 `Output`  ➤ [1]

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 **The number of the nodes in the tree is in the range `[0, 100]`.** </br>

🔹 **`-100 <= Node.val <= 100`** </br>

</br>
