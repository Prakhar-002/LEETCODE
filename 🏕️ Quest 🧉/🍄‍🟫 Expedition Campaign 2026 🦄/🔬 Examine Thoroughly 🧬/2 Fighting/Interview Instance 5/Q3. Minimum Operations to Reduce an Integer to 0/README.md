# Description 📜 ˋ°•*⁀➷

### You are given a positive integer `n`. In one operation, you may **add or subtract a power of 2** (i.e., a number of the form `2^i` for `i >= 0`) from `n`.

### Your task is to return the **minimum number of operations** needed to make `n` equal to `0`.

</br>

# Example 💡 1️⃣ ˋ°•*⁀➷

### 📥 `Input`  ➤ `n = 39`

### 📤 `Output`  ➤ `3`

### 🔦 `Explanation`  ➤  
- Add `2^0 = 1` → `n = 40`  
- Subtract `2^3 = 8` → `n = 32`  
- Subtract `2^5 = 32` → `n = 0`  
It can be shown that `3` operations is the minimum.

</br>

# Example 💡 2️⃣ ˋ°•*⁀➷

### 📥 `Input` ➤ `n = 54`

### 📤 `Output`  ➤ `3`

### 🔦 `Explanation`  ➤  
- Add `2^1 = 2` → `n = 56`  
- Add `2^3 = 8` → `n = 64`  
- Subtract `2^6 = 64` → `n = 0`  
So the minimum number of operations is `3`.

</br>

# Constraints 🔒 ˋ°•*⁀➷

🔹 `1 <= n <= 10^5` </br>

</br>
